Atlas Course Advisor System

Type: Deployed Academic App

Date: 2024

Location: Durham, North Carolina

The Atlas Course Advisor System enhances Duke University's course advising process with a conversational AI chatbot. The chatbot offers 24/7 support, leveraging semantic search to provide students with personalized course recommendations, resources, and requirements. This project empowers students to optimize their academic experience at Duke.

AI Integration

  • Retrieval-Augmented Generation (RAG): Enhances LLM accuracy by supplying relevant course data.
  • OpenAI LLM Models: Uses GPT-3.5-turbo and GPT-4 to generate contextually-aware responses.
  • Phi-3-Mini Model: Routes queries to appropriate databases with Microsoft’s compact language model.
  • Fine-Tuning & Prompt Engineering: Improves chatbot accuracy through refined prompts and examples.

Key Features

  • AI Chatbot: Conversational chatbot for course advising questions.
  • User Survey: Collects optional data to personalize chatbot responses.
  • Course Quick View & Skill Tags: Displays course details and skill tags within the chat.
  • Kubernetes Deployment: Web app deployed on OKD and Azure for seamless access.

Future Directions

Graph Database Integration: Migrating to a graph database for better understanding of complex course sequences and relationships.

Expanded Deployment: Scaling the project to other universities in North Carolina through NCShare.
